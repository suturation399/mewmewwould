# mewmewworld
喵的腦洞世界

不建議看代碼看能力，都是很暴力的東西

感謝T兄，那不勒斯冰淇淋，Sakisan師傅，還有最重要的喵打老大！！

所有技能都板模型式調數值，也不是很難做的技能

看名字就知道這3位也是參考其它遊戲做出來的，也感謝那些遊戲

本喵咪是這方面的新手，請多多指教，也請各位大佬（特別是以上4位）多多提點

## Velhari
> Velhari是一位有野心并且野蠻兇殘的世青賽德國代表隊中堅  

+ 綜述：
	+ Velhari在開局時啟用抑制光環。
	+ 東四局開始時，維哈里會把她激活的光環換為蔑視光環。
	+ 南三局開始時，維哈里再次將其激活的光環換為怨恨光環。
	+ 光環效果在階段轉換時不會消失，只會隨對局數持續減弱效果。
	
+ 第1階段：壓迫
	+ 壓迫光環：釋放詛咒光環，使他家配牌摸牌都變弱。該效果會隨對局數持續增強。

+ 第2階段：蔑視
	+ 蔑視光環：釋放詛咒光環，降低他家摸進寶牌的機率。該效果會隨對局數持續增強。
	+ 灼燒燃炎：提升自家摸赤寶牌的機率。
	
+ 第3階段：惡意
	+ 惡意光環：釋放詛咒光環，使四家配牌摸牌都變強。該效果會隨對局數持續增強。
	+ 暴君壁壘：為自身上一層防禦性壁壘，降低摸到銃牌的機率。
	+ 腐化洗禮：用黑暗魔法強化自身，使對銃牌產生感知。

## Jaina
> Jaina Proudmoore是世青賽英國代表隊大將

+ 匯聚暴風雪
	+ Jaina每次摸牌都會獲得一定的暴雪能量和冷凍能量。
	+ 冰冷的暴風雪會使周圍溫度降低,並同時限制進章能力。
	+ 暴雪能量50以下時，他家手牌必須有1張dora，不然進章會被凍結。
	+ 暴雪能量50後，他家手牌必須有3張dora，不然進章會被凍結。
	+ 暴雪能量70後，他家手牌必須有5張dora，不然進章會被凍結。

+ 第1階段：冰封之怒
	+ 冰之環：冷凍能量100後，他家進章會受限，效果直到Jaina回到低能量。

+ 第1.5階段：尖嘯之風
	+ 棱彩镜像：Jaina的幻象碎片使Jaina能量循環速度提高3倍。
	+ 冰封之怒：第1階段的所有技能。
	
+ 第2階段：海洋之女
	+ 冰寒凍土：Jaina的法術凍結整個場地，使全體越來越容易摸到冷色的牌。
	+ 寒冰裂片：自家進章會根據冷凍能量而增強。

+ 第2.5階段：冰霜速凍
	+ 結晶塵暴：大幅提升自家配牌獲得白的機率。
