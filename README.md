## Velhari
> Velhari是一位有野心并且野蠻兇殘的世青賽德國代表隊王牌中堅  

+ 綜述：
	+ Velhari在開局時啟用抑制光環。
	+ 東四局開始時，維哈里會把她激活的光環換為蔑視光環。
	+ 南三局開始時，維哈里再次將其激活的光環換為怨恨光環。
	+ 光環效果在階段轉換時不會消失，只會隨對局數持續減弱效果。
	
+ 第1階段：壓迫
	+ 壓迫光環：釋放詛咒光環，使他家配牌摸牌都變弱。該效果會隨對局數持續增強。

+ 第2階段：蔑視
	+ 蔑視光環：釋放詛咒光環，降低他家摸進寶牌的機率。該效果會隨對局數持續增強。
	+ 灼燒燃炎：提升自家摸赤寶牌的機率。
	
+ 第3階段：惡意
	+ 惡意光環：釋放詛咒光環，使四家配牌摸牌都變強。該效果會隨對局數持續增強。
	+ 暴君壁壘：為自身上一層防禦性壁壘，降低摸到銃牌的機率。
	+ 腐化洗禮：用黑暗魔法強化自身，使對銃牌產生感知。

## Jaina
> Jaina Proudmoore是世青賽英國代表隊成員  

+ 貫穿全場的機制：匯聚暴風雪
	+ Jaina每次摸牌都會獲得一定的暴雪能量和冷凍能量。
	+ 冰冷的暴風雪會使周圍溫度降低,並同時限制進章能力。
	+ 暴雪能量50以下時，他家手牌必須有1張dora，不然進章會被凍結。
	+ 暴雪能量50後，他家手牌必須有3張dora，不然進章會被凍結。
	+ 暴雪能量70後，他家手牌必須有5張dora，不然進章會被凍結。
	+ 暴雪能量100後，進章會嚴重受限，效果直到Jaina回到低能量。
	
+ 第1階段：冰封之怒
	+ 冰之環：冷凍能量100後，他家進章會嚴重受限，效果直到Jaina回到低能量。

+ 第1.5階段：尖嘯之風
	+ 迅猛旋風：提升自家摸到四風的機率。
	+ 棱彩镜像：Jaina的幻象碎片使Jaina能量循環速度提高5倍。
	+ 冰封之怒：第1階段的所有技能。
	
+ 第2階段：海洋之女
	+ 冰寒凍土：Jaina的法術凍結整個場地，使全體越來越容易摸到冷色的牌。
	+ 寒冰裂片：自家進章會根據冷凍能量而增強。

+ 第2.5階段：冰霜速凍
	+ 結晶塵暴：大幅提升自家配牌獲得白的機率。
	+ 海洋之女：第2階段的所有技能。

## Dia
> Dia Darkwhisper是世青賽牙買加代表隊成員
> designing and testing
	
+ 虛無薄霧
	+ Dia的黑暗力量以黑暗薄霧籠罩牌山，減少他家對牌山的影響。

+ 夢魘幻貌
	+ Dia每次摸牌都會獲得一定的瘋狂能量。
	+ 能量100後，Dia會變成的魔幻面貌，提升進章能力，效果直到Dia回到低能量。

+ 哀嚎之懼
	+ Dia聽牌後讓黑暗遮蔽整個空間。
	+ 使他家聽牌後使黑暗凝聚，他家容易摸到自家銃牌。
  
 ## Zul
> Zul是世青賽南非代表隊成員  
> 她是南非議會領導者，國家的最高顧問，並被讚頌為最偉大的預言者的女兒  
> 如今，她打算帶領王國從麻雀方面稱霸世界  

+ 黑暗啟示
	+ 門前清狀態時，當有人進入1向聽或以下，Zul便會從血神得到啟示，知道其向聽數和有效牌

+ 腐化軌跡
	+ 門前清狀態時，Zul會持續受到血神的強化，摸牌會隨對局數持續變強。
 
+ 腐化之血
	+ 鳴牌後，失去黑暗啟示和腐化軌跡，並且摸牌會變弱。
