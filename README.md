## Azshara
> (測試製作中)Azshara女王是世青賽亞特蘭提斯代表隊的王牌大將  
> 身為地球有史以來最強大的帝國之一，亞特蘭提斯的女王。  
> Azshara計劃和密謀了數千年，為這個命運的時刻做準備。   
> 現在，隨著所有的布局都已經到位，終於到她揭開底牌的時候了。  

+ 泰坦規劃
	+ 遠古結界（其他三家牌河）：三個遠古結界可以保護牌局正常進行，但必須為其充能。
	+ 壓力驟增：隨著每個遠古結界的能量斷絕，空間會受到週期性的壓力驟增，使家配牌變差，而失去能量的遠古結界再也不能重啟。
	+ 災難事故：如果三個遠古結界（其他三家牌河）均失去全部能量，將會迎來災難性事故，所有他家都會敗陣。
	+ 力量結界（Azshara牌河）：Azshara能量之泉為她所用。只要力量結界獲得能量，Azshara進章就會增強。
	+ 汲取遠古結界：Azshara汲取其他遠古結界的能量以提升力量結界的能量。

+ 結界能量機制補充
	+ 充能機制：其他三家牌河的每張捨牌都會計算進該結界的能量，每張的能量值根據牌的數字。
	           例如9p9s9m會充9點能量，3p3s3m則3點能量，一切字牌都不能為結界充能。
	+ 汲取機制：Azshara牌河的每張捨牌都會計算汲取結界的能量，每張的能量值根據牌的數字。
	           例如3p3s3m會從他家3個遠古結界都汲取3點能量，並為力量結界充能9點能量。
	+ 能量機制：每局配牌時，3個遠古結界能量都會由30點開始，之後跟隨當局牌河而改變。但如能量己完全斷絕，則不會回復。力量結界能量每局由0點開始。
	
+ 女王法令
	+ 隨機對牌山發布了不同種類的法令，他家可以根據進章猜測法令，根據她的法令會更容易做牌。
	+ 服從：女王命令牌山，使你更容易摸進他家的有效牌。
	+ 受苦：女王命令牌山，使你摸不到寶牌。
	+ 集合：女王命令牌山，使你更容易一直摸進同一色的牌。
	+ 獨立：女王命令牌山，使你更容易保持手牌萬筒索三種色的數量平衡。
	+ 前進：女王命令牌山，使你更難以摸到自己牌河上的牌。 
	+ 停留：女王命令牌山，使你更容易摸到自己牌河上的牌。
